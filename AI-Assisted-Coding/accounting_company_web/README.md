# Minev — Сайт за счетоводна кантора

Прост и отзивчив статичен сайт за счетоводна фирма. Съдържа следните страници:
- `index.html` — Начало
- `about.html` — За нас
- `prices.html` — Цени
- `offers.html` — Оферти
- `contact.html` — Контакти

Как да стартирате локално:

1. Отворете файловете директно в браузъра (двуклик върху `index.html`) или стартирайте прост HTTP сървър:

```pwsh
python -m http.server 8000
```

2. Посетете `http://localhost:8000/index.html` или пътя към папката.

Бележки:
- Заменете изображението в `assets/logo.png` и `assets/favicon.png` със собствените ви файлове.
- Формата в `contact.html` е клиентска (action="#"); свързването с backend изисква допълнителна конфигурация.
